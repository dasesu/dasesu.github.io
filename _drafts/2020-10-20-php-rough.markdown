Estos apuntes no tienen el proposito de explicar ni transmitir las cosas "bonito" ni en orden, sólo de ser una rápida referencia comandos y funcionalidades.

Alternar entre diferentes versiones de php instaladas en el sistema:
---
Interactive switching mode
```
sudo update-alternatives --config php
```
y seleccionamos la versión que queremos usar.

Podemos hacerlo u poco mas directo 
```
$ sudo update-alternatives --set php /usr/bin/php7.1
```

Si queremos hacerlo de una forma manual, directamente en la configuración de apache:
```
$ sudo a2dismod php5.6           # deshabilita php5.6
$ sudo a2enmod php7.1            # habilita php7.1
$ sudo service apache2 restart   # reiniciamos apache
```

Crear un servidor php
---
crea un servidor en el directorio actual
```
php -S localhost:4000
```

Etiquetas php
---
Php es un lenguaje fuertemente acomplado a html, basicamente adopta la estructuara de html, y podemos usar etiquetas <?php y ?> para escapar o iniciar codigo php.
```php
<?php
echo "Hello World!";
?>
```

Comando Echo
---
Es un comando que nos permite escribir código html desde php, podría verse como una forma de imprimir por pantalla desde php, similar a lo que en C++ sería printf. Veamos algunos usos.
```php
echo "string";
echo $variable;
echo "el valor de la variable es".$variable;
echo "<p>el valor es".$variable."y el texto continua</p>";
```

Tipos de datos
---
Las variables en php se definen con el prefijo `$`, y pueden ser del tipo String, entero, flotante, booleano, el tipo de dato se define con la forma en las escribimos, es decir no indicamos explicitamente el tipo de dato, y una variable puede pasar a ser de un tipo de dato a otro en una nueva asignación, es decir es unlenguaje debilmente tipado.

Un String va entre comillas  
Un entero es una asignación numerica sin decimales  
un flotante es una asignación numerica con decimales y los decimales se indican con un punto  
un flotante puede tomar el valor de true o false, tambien 1 y 0
```php
$mystring = "This is my string";
$myInteger = 42;
$myFloat = 3.14;
$myBool=true;
```
